# AlipayForest
支付宝蚂蚁森林

文件说明：
AlipayForest.js中文版本的蚂蚁森林
AlipayForestUK.js英文版蚂蚁森林

1、把autojs.4.0.1 Beta.apk安装到手机上面

2、打开Auto.js应用程序，开启无障碍服务。

3、点击右下角“+”，点击“导入”脚本文件，AlipayForest.js或者AlipayForestUK.js文件。

4、点击运行，第一次使用需要确认赋予截屏权限。

5、脚本将会自动打开支付宝-蚂蚁森林页面收取能量，注意需要将蚂蚁森林放到支付宝首页。

6、收取完成后自动返回支付宝首页。

7、若中途需要关闭可在Auto.js中设置“音量上键停止所有脚本”。

8、早上7:10到7:40开启脚本可在此时间段内不断循环收取。
